# Example Repo

This repo shows how to provide data via git to https://api.vitrine.gesis.org

1. The repo must be available over HTTPS and readable for everyone
2. The repo must contain a `data` directory (name == 'data')
3. The data directory must contain only xml files that are valid against http://www.da-ra.de/dara/schemadefinitions/dara.xsd[da|ra Metadata Schema Version 4.0]
